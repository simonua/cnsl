Install App

Add the web app to your Home Screen for quick access to pools, teams, and meets. On a phone or tablet, the matching instructions open automatically.

Android ▲

Open this page in a web browser.

  1. Open the browser menu.
  2. Choose Install app or Add to Home screen.
  3. Follow the browser prompt to finish.
Apple ▲

Use these steps on an iPhone or iPad.

  1. Open this page in Safari.
  2. Tap the Share button in the browser toolbar.
  3. Choose Add to Home Screen.
  4. Tap Add to finish.
